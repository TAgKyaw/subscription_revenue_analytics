# Project Overview

A Revenue Dashboard building analysis project for a Subscription-based D2C Business Model using zero-party collected customer data to identify subscription trends, revenue acceleration and monthly growth. Simulated for pet food market with a brand analysis with synthetic data.

# Project Structure

- **requirements.txt** : Installing required libraries from requirements.txt
- **butternutbox.db** : Database created to start with EDA and conduct analysis. Also available to build own schema using:
    - setup.py
    - extend_database.py
- **setup.py** : Synthetic Data Generation for randomised customer data with modelled data schema
- **extend_database.py** : Extended data features updated in the database.
- **notebook** : all EDA, data preparation and business analytics in Python and SQL.
- **duckdb** : duckdb database is used for quick prototyping and analysis.
- **images** : revenue dashboard visuals
- **Slides** : Portfolio Slides presenting the revenue dashboard built.
